# Password Strength Analyzer

## Description
A Java-based Password Strength Analyzer that evaluates the strength of a password based on:
- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Common password detection

## Features
- Password strength analysis
- Security score calculation
- Password improvement suggestions
- Strong password generation

## Technologies Used
- Java
- VS Code

## How to Run

Compile the program:

```bash
javac PasswordStrengthAnalyzer.java
```

Run the program:

```bash
java PasswordStrengthAnalyzer
```
